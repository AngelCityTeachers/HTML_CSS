This is level 5 heading. When you give them heading style tags, they will probably want to write more, so they
will ask for paragraph tags.

When they are finished writing content within paragraph tags, they'll want to use a break tag to break the line like a return on a keyboard. Then they will type and type
all kinds of things, and will want to view them in the browser. Anything within the body of a web page is displayed in the main browser window.

A paragraph consists of one or more sentences that form a self contained unit of discourse. The start of a paragraph is
indicated by a new line.

To make sure you see some very important words, they might want to bold them with the the bold tags. This is how we make a word appear <b></ bold>.

When they are done bolding words, they will want to use italics.

They might get carried away and forget to structure the page to make the content easier to read, so they will ask for elements such as lists to be ordered or unordered.

Text is easier to understand when it is split up into units of text.
